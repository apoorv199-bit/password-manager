# 🔐 Password Manager

Just me building a password manager from scratch in Go to understand how things actually work under the hood.

## Why this exists

I wanted to go beyond just using tools and actually learn:

- how passwords are stored securely
- how encryption works in real systems
- how vaults are designed
- how authentication flows are implemented

So yeah… building one myself.

## What I'm working on

- [ ] User authentication
- [ ] Secure password storage
- [ ] Encryption / Decryption
- [ ] Vault system
- [ ] Add / update / delete credentials
- [ ] Key derivation (Argon2)
- [ ] API layer
- [ ] Database integration

## Tech

- Go
- Some crypto stuff
- DB (still deciding)
- REST APIs

## Status

Still building 🚧  
Things might break. Probably will.

## Note

This is mainly a learning project.  
Not something you should trust with real passwords (at least for now 😅)
